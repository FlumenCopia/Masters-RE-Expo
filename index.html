03 — Signature Feature
The Pre-Registration + Calendar Reminder System

The biggest drop-off in expo attendance: people see the ad, get interested, intend to come — and forget. This system eliminates that.

💡 Anoop's Idea — Recommended for Implementation
"I'm Coming" → Auto Calendar Event → Reminder Chain
When a user clicks a pre-registration link (from any ad, post, story, or WhatsApp message), they land on a simple page with one choice: "Yes, I'm visiting Masters Expo 2026." One click automatically adds the event to their Google Calendar / Apple Calendar with a location pin (LuLu Mall) and sends 3 automated reminders: 7 days before, 2 days before, morning of. This converts casual interest into committed attendance.
1User sees ad/post → clicks pre-registration link
2Lands on simple "I'm Coming" page — name + phone only
3One click → Google/Apple Calendar event added automatically
4Automated WhatsApp reminder: 7 days, 2 days, morning of event
5Masters gets a live count of pre-registered visitors
🛠 How to Build This
Simple landing page (1 screen) with Add to Calendar button using AddEvent.com or Calendly integration. Form data (name + phone) captured in Google Sheets or Airtable. WhatsApp reminders via Interakt or AiSensy (both WABA-compliant, work in India). Build time: 2–3 days. Cost: near-zero (most tools have free tiers sufficient for this scale).
AddEvent.com AiSensy / Interakt Google Sheets Airtable
📈 Why This Converts
Industry data shows that event reminder sequences increase actual attendance by 25–40% vs unreminded registrants. For a free-entry expo with 15K–20K footfall target, converting even 2,000 pre-registrations to actual visits using this system is a significant uplift. It also gives Masters a pre-event attendance count they can share with sponsors as proof of demand — before the expo even opens.
